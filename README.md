## Apache Beam Word Count Lab

- A Python implementation of a word counting pipeline using Apache Beam, designed to analyze text files (like Shakespeare's plays) and generate word frequency statistics.

- This project demonstrates the use of Apache Beam for processing large text files to Count word frequencies, Generate statistics about word usage

- The pipeline is optimized for Windows environments and includes automatic cleanup of previous outputs.

#### Installing Apache Beam

pip install apache-beam
